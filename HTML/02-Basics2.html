On CSS

Backgrounds

You can set the background of your webpage to a simple color, or you can use an image. The background you set can be for the entire page and/or for certain divs.

Width and Height

The width and height properties are used to set the width and height of an element

The default setting for width and height is auto, which means the browser decides the width and height.

Other settings for width and height are:

  • specified length values such as px and em
  • percent of the containing div (ex: 50%, 100%)

Border Style

The border-style property can take up to four values (top border, right border, bottom border, and left border). The following border-style values are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove: 3D grooved border, depends on border-color value
  • ridge: 3D ridged border, depends on the border-color value
  • inset: 3D inset border, depends on the border-color value
  • none: no border
  • hidden: hidden border
  • mixed: you can set a different border style for the top, bottom, right, and left

Border Width

VERY IMPORTANT: border-width will not work unless you have border-style set!!

Border-width can be set to a specific size (px, em, etc) or you can use one of the following pre-set settings: thin, medium, thick. The border-width property can also take up to four values (top, right, bottom, left) (although I can't seem to get this to work)

Border Color

VERY IMPORTANT: border-color will not work unless you have border-style set!!

Border-color can be set by the following ways:

  • name: specify a color such as "lightblue"
  • Hex: specify a hex value, like "#eee"
  • RGB: specify a RGB value, like "rgb(100,100,100)"
  • transparent

Position

The position property specifies the type of positioning method used for an element. The four position values are:

  • fixed: this will pin something to a spot, and it will stay there as you scroll (ex: navigation bar). Fixed position takes elements out of the normal document flow.
  • static: this is the default setting, it is not affected by top, bottom, left, and right properties. It is always positioned according to the normal flow of the page.
  • relative: does not take the element out of normal document flow. Still appears within its space/div/container, but allows it to be offset a bit from the top, bottom, right, left. Top, right, bottom, and left properties will move it away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
  • absolute: takes element out of the normal document flow. If a parent element position is set to relative, and the child element is set to absolute, the child element's new (0,0) coordinate will be the upper left hand corner of the parent element instead of the whole page.
  • The z-index property allows you to override the normal stacking order of the page. The default z-index is 0, and you can set that value to be positive or negative. An element with a greater z-index will stack in front of an element with a lower z-index. If two elements have the same z-index, or don't have one assigned, the element positioned last in the HTML will show on top. You must have a position assigned in order to assign a z-index.
